# Vitiate How-To Guide (No Console Required)

## Install in Chrome

1. Open the Vitiate repository page on GitHub.
2. Click **Code** → **Download ZIP**.
3. Extract the ZIP file on your computer.
4. Open Chrome and go to `chrome://extensions/`.
5. Turn on **Developer mode** (top-right).
6. Click **Load unpacked**.
7. Select the extracted `chrome-extension/` folder.
8. Vitiate should now appear in your extensions list and toolbar.

## Use Vitiate

1. Open any website.
2. Click the Vitiate extension icon.
3. Use the popup controls to:
   - Enable/disable protection globally
   - Enable/disable protection for the current domain
   - Adjust poisoning intensity
   - Toggle defense modules
4. Close the popup — settings are saved automatically.

## Update to a New Version

1. Download the latest repository ZIP from GitHub.
2. Extract it.
3. In `chrome://extensions/`, find Vitiate and click **Remove** (or **Reload** after reselecting folder).
4. Click **Load unpacked** and select the new `chrome-extension/` folder.
